# PGN Visualizer

A clean and elegant local web app for reading, creating, merging, and analyzing chess PGN files.

## Features

- Load one or multiple PGN files locally
- Browse full game headers and move lists
- Create new PGN files from a built-in form
- Merge multiple PGNs and organize games by date
- View dynamic statistics for performance, results, openings, and recent games
- Responsive dark/gold interface designed for comfortable reading

## Tech Stack

- HTML
- CSS
- JavaScript
- Chart.js

## Getting Started

1. Clone or download this repository
2. Open the HTML file in your browser

```bash
git clone https://github.com/Toporoski/PGN-visualization.git
```

Then open:

```text
index.html
```

## Usage

- Upload one or more `.pgn` files to inspect games
- Switch between the viewer, PGN creator, and statistics tabs
- Use the merge option to combine multiple PGNs sorted by date
- Export generated PGN files directly from the interface
